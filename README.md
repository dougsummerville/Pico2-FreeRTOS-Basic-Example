# Pico2-FreeRTOS-Basic-Example
A basic working example for configuring and using FreeRTOS on the RP Pico2

Basic Steps 

-https://github.com/dougsummerville/Pico2-FreeRTOS-Basic-Example.gitcd ~
-git clone https://github.com/raspberrypi/pico-sdk.git
-cd pico-sdk
-git submodule update --init
-cd ~
-git clone https://github.com/raspberrypi/FreeRTOS-Kernel.git
-git clone https://github.com/dougsummerville/Pico2-FreeRTOS-Basic-Example.git
-cd Pico2-FreeRTOS-Basic-Example
-cmake -B build && cmake --build build
